-HTML
-CSS

##Functionality
-Add/Delete Task
-Mark task as "completed"/"not completed"
-Filtering tasks
-Counter of "all tasks"/"completed tasks"/"not completed tasks"
-Display date and time of task creation

##Script for launching
```
npm run start
```
